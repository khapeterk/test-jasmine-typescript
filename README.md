# test-jasmine-typescript

> :ledger: Test project with Jasmine & TypeScript

## Setup
If using npm
```
$ npm install
```

If using yarn
```
$ yarn
```

## Unit tests
If using npm
```
$ npm test
```

If using yarn
```
$ yarn test
```
